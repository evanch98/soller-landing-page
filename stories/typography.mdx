# Typography

The `fontSize`, `fontWeight`, `lineHeight`, and `letterSpacing` for each typeset are already defined in the `tailwind.config.ts` file. For example, `text-h1` means `h1: ["72px", { fontWeight: 800, lineHeight: "110%" }]`. We don't need to use `text-bold` for the `fontWeight`.

**Font:** Roboto

**Weights:** 300(thin), 400(regular), 500(medium), 700(bold), 800(extrabold)

  fontSizes={[
